# Press Start!

**Developer:** Omar Al Salti  
**Website link:** https://press-start-now.vercel.app/

---

## Overview

**Press Start** is a project website that explores the history of video games, from classic consoles to iconic creators who shaped the industry.  
The project aims to be fun, informative, and visually engaging!

---

## Website Structure

The website contains multiple themed pages, each covering a specific aspect of gaming history:

- **Home** – Introduction to the website  
- **Timeless Games** – Iconic games that defined generations  
- **Console Legacy** – Evolution of major gaming consoles  
- **Pixels on the Go** – Handheld and portable gaming  
- **Retro Evolve** – A timeline of Gaming Evolution  
- **Game Geniuses** – Influential developers and creators  
- **Fun Facts** – Interesting facts about games and consoles  
- **Game Challenge!** – Simple interactive game activity  
- **Game Dictionary** – Basic gaming terms and definitions  
- **Mini Quiz** – Small quiz page with alert-based answers  
- **Emoji Game** – Guess game titles using emoji clues  
- **About Us** – Project and Developer informations  
- **Contact Us** – Contact form for user feedback

---

## Technical Implementation

This project is built using:

- **HTML** – Page structure  
- **CSS** – Styling and layout  
- **JavaScript** – Simple interactions (quiz, guessing games, alerts)  

---

## Features

- Simple and easy to interact design
- Navigation across all pages for easy switching
- Interactive JavaScript quizzes and guessing games  
- Videogames themed styling and layouts
